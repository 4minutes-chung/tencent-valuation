# Project Memory

Last updated: 2026-04-02

## Operating Priority

1. Focus deliverables on story quality, investment narrative clarity, graphs, and visuals.
2. Treat additional modeling or code changes as secondary unless explicitly requested.
3. Prefer outputs that are presentation-ready for non-technical readers.

## Execution Defaults

1. Lead with chart-first communication (what changed, why it matters, what to do).
2. Keep technical detail in appendices, not the main narrative.
3. Optimize reports for decision-making speed and readability.
